# Simple-To-Do-List-Manager

โปรแกรมจัดการรายการสิ่งที่ต้องทำ (To-Do List)  เป็นโปรแกรมรันบน Google Colab เพื่อช่วยให้ผู้ใช้สามารถจัดการกิจกรรมประจำวันได้โดยตรงจากเทอร์มินัลอย่างง่ายดาย

มินิโปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชา **CP352301 Script Programming** ประจำภาคเรียนที่ 1 ปีการศึกษา 2568


# **ความคืบหน้าของแต่ละสัปดาห์**

# Sprint 1 (Week1)

**เป้าหมาย**

1.สร้างโครงสร้างพื้นฐานของโปรแกรม To-Do List ✅

2.พัฒนาฟังก์ชันหลักให้ครบถ้วน: เพิ่มรายการ (Add),ลอบรายการ (Delete), ดูรายการทั้งหมด (View), และ ทำเครื่องหมายว่าเสร็จสิ้น (Mark as complete) ✅

3.ทำให้โปรแกรมมีความทนทานต่อการใช้งาน โดยการจัดการข้อผิดพลาด (Error Handling) จาก Input ของผู้ใช้ ✅

4. โปรแกรมสามารถเริ่มต้นและจบการทำงานได้อย่างสมบูรณ์ ✅

# Sprint 2 (Week2)

**เป้าหมาย**

ขยายขีดความสามารถของโปรแกรม: เพิ่มฟังก์ชันแก้ไขรายการ (Edit) ✅

เพิ่มฟังก์ชันการค้นหา: ผู้ใช้สามารถค้นหารายการที่ต้องการจากคำอธิบาย (description) ได้ ✅

ปรับปรุง UI (User Interface): ปรับการแสดงผลให้เป็นระเบียบและเป็นมิตรกับผู้ใช้มากขึ้น ✅

เชื่อมต่อกับ GitHub: สร้าง Pull Request (PR) เพื่อรวมโค้ดที่พัฒนาใน Sprint นี้เข้ากับโค้ดหลัก ✅

# Sprint 3 (Week3)

**เป้าหมาย**

ปรับโครงสร้างโปรแกรม (Code Refactoring): แยกฟังก์ชันทั้งหมดออกจาก main() เพื่อให้โค้ดสะอาดและจัดการง่ายขึ้น ✅

สร้างไฟล์สำหรับ Unit Test: เขียน Unit Test สำหรับฟังก์ชันหลักอย่าง add_task และ delete_task เพื่อตรวจสอบความถูกต้องของการทำงานโดยอัตโนมัติ ✅

เพิ่มการจัดการข้อมูล: เพิ่มฟังก์ชันในการบันทึกและโหลดรายการงานจากไฟล์ เพื่อให้ข้อมูลไม่หายไปเมื่อปิดโปรแกรม ✅
